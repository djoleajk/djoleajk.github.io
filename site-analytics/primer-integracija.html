📊 Kako integrisati Site Analytics Tracker

🎯 Cilj: Dodati Analytics praćenje na vaš postojeći sajt u 2 minuta.

Korak 1: Dodajte tracker script

Na kraju svake stranice, pre zatvaranja </body> taga, dodajte:

vaš-sajt.html
<!DOCTYPE html> <html> <head> <title>Vaš sajt</title> </head> <body> <!-- Vaš sadržaj --> <h1>Dobrodošli na moj sajt</h1> <p>Neki sadržaj...</p> <!-- 🔥 Dodajte ovu liniju na kraju --> <script src="analytics-tracker.js"></script> </body> </html>
✅ I to je sve! Tracker će automatski početi da beleži posete.

Korak 2: Dodajte tracker na SVE stranice

Ako imate više stranica (home.html, about.html, contact.html...), dodajte istu liniju na svaku:

vaš-sajt/ ├── index.html ← dodaj script ├── about.html ← dodaj script ├── contact.html ← dodaj script ├── products.html ← dodaj script └── analytics-tracker.js

Korak 3: Postavite Dashboard

Stavite index.html (Analytics Dashboard) u poseban folder:

vaš-sajt/ ├── home.html ├── about.html ├── analytics-tracker.js └── analytics/ ├── index.html ← Dashboard ├── script.js └── style.css

Ili jednostavno preimenujte Dashboard u analytics.html:

vaš-sajt/ ├── index.html ← Vaša početna ├── about.html ├── analytics.html ← Dashboard ├── analytics-tracker.js ├── script.js └── style.css

Primer: WordPress sajt

⚠️ Za WordPress, dodajte tracker u footer.php fajl vašeg theme-a.
wp-content/themes/vaš-theme/footer.php
<footer> <!-- Footer sadržaj --> </footer> <!-- 🔥 Dodajte pre zatvaranja body taga --> <script src="<?php echo get_template_directory_uri(); ?>/analytics-tracker.js"></script> <?php wp_footer(); ?> </body> </html>

Napredne opcije

1. Custom eventi

Pratite specifične akcije (npr. klikove na dugmad):

<button onclick="trackButtonClick()">Pošalji poruku</button> <script> function trackButtonClick() { if (window.analyticsTracker) { window.analyticsTracker.trackEvent('button_click', { button: 'Pošalji poruku', timestamp: new Date().toISOString() }); } } </script>

2. Praćenje forme

Beležite kada korisnik pošalje formu:

<form onsubmit="trackFormSubmit(event)"> <input type="email" name="email" required> <button type="submit">Prijavi se</button> </form> <script> function trackFormSubmit(event) { if (window.analyticsTracker) { window.analyticsTracker.trackEvent('form_submit', { form: 'Newsletter signup', timestamp: new Date().toISOString() }); } } </script>

3. Praćenje scroll dubine

Pratite koliko korisnici scrolluju:

<script> let maxScroll = 0; window.addEventListener('scroll', () => { const scrollPercent = Math.round( (window.scrollY / (document.body.scrollHeight - window.innerHeight)) * 100 ); if (scrollPercent > maxScroll) { maxScroll = scrollPercent; if (window.analyticsTracker && (scrollPercent === 25 || scrollPercent === 50 || scrollPercent === 75 || scrollPercent === 100)) { window.analyticsTracker.trackEvent('scroll_depth', { depth: scrollPercent + '%' }); } } }); </script>

Provera da li radi

  1. Otvorite stranicu vašeg sajta
  2. Pritisnite F12 (Developer Tools)
  3. U Console tab-u videćete:
    ✅ Analytics Tracker aktiviran - Praćenje u toku!
  4. U Application → Local Storage videćete podatke
🎉 Gotovo! Vaš sajt sada ima potpuno funkcionalni analytics sistem.

Šta dalje?

  • 📊 Otvorite Dashboard (analytics.html ili index.html)
  • 🔍 Pratite posete u realnom vremenu
  • 📥 Eksportujte podatke redovno
  • 🎨 Prilagodite Dashboard svom dizajnu
  • 🚀 Dodajte backend za perzistenciju podataka
💡 Pro savet: Za produkciju, razmislite o slanju podataka na server umesto čuvanja samo u localStorage. Tako ćete imati centralizovane podatke za sve posetioce.
